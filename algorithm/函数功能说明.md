## 1. unwrap/ — 结构光相位解包与质量评估

本模块实现多频时间相位解包算法的核心计算步骤，包括包裹相位计算、多频合成、时间展开与质量筛选。

---

### 1.1 CalculatePhaseMatrix

| 项目           | 内容                                        |
| -------------- | ------------------------------------------- |
| **文件** | `unwrap/CalculatePhaseMatrix.m`           |
| **类型** | 函数                                        |
| **用途** | 对每个频率的 N 步相移图像组计算包裹相位主值 |

**函数签名**

```matlab
phase = CalculatePhaseMatrix(images, N)
```

**输入**

| 参数       | 维度           | 说明                                            |
| ---------- | -------------- | ----------------------------------------------- |
| `images` | `cell(3, N)` | 3 个频率各 N 步相移图像；每个元素为灰度图像矩阵 |
| `N`      | 标量           | 相移步数（通常为 4）                            |

**输出**

| 参数      | 维度           | 说明                                          |
| --------- | -------------- | --------------------------------------------- |
| `phase` | `cell(1, 3)` | 三个频率对应的包裹相位图，值域$[-\pi, \pi]$ |

**算法**：N 步相移公式

$$
\phi = \arctan\!\left(\frac{-\sum_{i=1}^{N} I_i \sin\!\left(\frac{2\pi(i-1)}{N}\right)}{\sum_{i=1}^{N} I_i \cos\!\left(\frac{2\pi(i-1)}{N}\right)}\right)
$$

### 1.3 Multifrequency_phase

| 项目           | 内容                                          |
| -------------- | --------------------------------------------- |
| **文件** | `unwrap/Multifrequency_phase.m`             |
| **类型** | 函数                                          |
| **用途** | 计算两个频率包裹相位之差并映射到$[0, 2\pi)$ |

**函数签名**

```matlab
pha12 = Multifrequency_phase(pha1, pha2)
```

**输入**：两幅包裹相位图（矩阵），值域均为 $[-\pi, \pi]$
**输出**：差频包裹相位 `pha12`，值域 $[0, 2\pi)$，始终非负

**作用**：用于多频时间展开的"粗尺度"相位生成，等效波长为两频率等效波长之积除以它们之差。

### 1.4 UNwrapPhase

| 项目           | 内容                                       |
| -------------- | ------------------------------------------ |
| **文件** | `unwrap/UNwrapPhase.m`                   |
| **类型** | 函数                                       |
| **用途** | 基于整数倍关系的时间相位展开（多频联合法） |

**函数签名**

```matlab
unwrappedPhase = UNwrapPhase(keyPhase, multiple, wrappedPhase)
```

**输入**

| 参数             | 说明                                        |
| ---------------- | ------------------------------------------- |
| `keyPhase`     | 低频/合成粗尺度展开相位（已展开，单位 rad） |
| `multiple`     | 精细频率相对于粗频率的整数倍数              |
| `wrappedPhase` | 精细频率对应的包裹相位$[-\pi, \pi]$       |

**输出**：`unwrappedPhase`，精细频率展开后的绝对相位（连续单调）

**算法**：$\phi_{unwrapped} = \phi_{wrapped} + 2\pi \cdot \mathrm{round}\!\left(\frac{k \cdot \phi_{key} - \phi_{wrapped}}{2\pi}\right)$

### 1.5 PhaseQualityMask（原 Copy_of_Monntonicity_Connected_Domain_Seg1）

| 项目             | 内容                                                                   |
| ---------------- | ---------------------------------------------------------------------- |
| **文件**   | `unwrap/PhaseQualityMask.m`（重命名后）                              |
| **类型**   | 函数                                                                   |
| **用途**   | 多特征融合的相位有效区域掩膜生成，兼顾相位梯度、误差能量与调制度对比度 |
| **被调用** | `restruction/reconstruction_SNR.m`                                   |

**函数签名**

```matlab
[mask, datamodulationpic, datamodulation, meanpic] = PhaseQualityMask(Img, hDeformedImages, PHDeformed_1, PHDeformed_2, PHDeformed_3, N)
```

**输入**

| 参数                 | 说明                                         |
| -------------------- | -------------------------------------------- |
| `Img`              | 水平方向展开相位图（绝对相位，用于梯度计算） |
| `hDeformedImages`  | `cell(3, N)` 三频相移图像容器              |
| `PHDeformed_1/2/3` | 三个频率对应的包裹相位主值图                 |
| `N`                | 相移步数                                     |

**输出**

| 参数                  | 说明                                        |
| --------------------- | ------------------------------------------- |
| `mask`              | 二值有效掩膜（多特征 AND 融合后去小连通域） |
| `datamodulationpic` | 调制度对比度图（用作 SNR 权重）             |
| `datamodulation`    | 调制度总和值                                |
| `meanpic`           | 平均亮度图                                  |

**内部算法步骤**

1. 调用 `CalcPhaseErrorEnergy` 分别计算三频误差能量图，以 90th 百分位数为阈值生成误差掩膜
2. 调用 `CalcModulationContrast` 计算调制度对比度掩膜（阈值 0.2）
3. 计算展开相位的 4-邻域最大梯度，以 $2\pi$ 为阈值生成梯度掩膜
4. 三层掩膜取交集，去除小连通域（< 100 像素）
5. 边缘修补：对邻接已有效像素且调制度满足 90% 阈值的零值点进行迭代填充

### 1.7 CalcPhaseErrorEnergy（原 PMPGetErrorEnergy）

| 项目             | 内容                                                             |
| ---------------- | ---------------------------------------------------------------- |
| **文件**   | `unwrap/CalcPhaseErrorEnergy.m`（重命名后）                    |
| **类型**   | 函数                                                             |
| **用途**   | 计算每个像素的归一化相位重建误差能量（越低表示该处条纹质量越好） |
| **被调用** | `PhaseQualityMask`（内部调用）                                 |

**函数签名**

```matlab
mean_ErrorEnergy = CalcPhaseErrorEnergy(SourceImg, phaseprincipal, N)
```

**输入**

| 参数               | 说明                                    |
| ------------------ | --------------------------------------- |
| `SourceImg`      | `cell(1, N)`，单频 N 步归一化相移图像 |
| `phaseprincipal` | 对应频率的包裹相位图                    |
| `N`              | 相移步数                                |

**输出**：`mean_ErrorEnergy`，与理想 N 步余弦的归一化均方误差图

**公式**（对每个像素 $p$）：

$$
E_p = \frac{1}{N}\sum_{i=1}^{N}\left(\hat{I}_{p,i} - \cos\!\left(\phi_p - \frac{2\pi(i-1)}{N}\right)\right)^2
$$

其中 $\hat{I}_{p,i} = \dfrac{I_{p,i} \cdot 255 - A_p}{B_p}$，$A_p$ 为平均亮度，$B_p$ 为条纹调制度（调用 `CalcFringeModulation` 计算）。

---

### 1.9 CalcFringeModulation（原 PMPGetModulation）

| 项目             | 内容                                          |
| ---------------- | --------------------------------------------- |
| **文件**   | `unwrap/CalcFringeModulation.m`（重命名后） |
| **类型**   | 函数                                          |
| **用途**   | 计算条纹调制度幅值（N 步相移条纹对比度）      |
| **被调用** | `CalcPhaseErrorEnergy`（内部调用）          |

**函数签名**

```matlab
modulationpic = CalcFringeModulation(SourceImg, N)
```

**公式**：

$$
B_p = \frac{2}{N}\sqrt{\left(\sum_{i=1}^{N} I_{p,i}\sin\frac{2\pi i}{N}\right)^2 + \left(\sum_{i=1}^{N} I_{p,i}\cos\frac{2\pi i}{N}\right)^2}
$$

注意：此处累加从 $i=1$ 开始（相位偏移 $2\pi/N$），与 `CalculatePhaseMatrix` 中从 $i=0$（即 $i-1$）开始的约定略有不同，二者均为标准 N 步公式的等效形式。

## 4. Chip/four/ — QFP 芯片引脚定位与共面度检测

本模块实现 QFP（四方扁平封装）芯片引脚的完整检测流程，分为二维图像引脚分割（Step 1-5）和三维点云共面度测量（Step 1-4）两个子管道。

**整体调用关系**

```
输入：灰度图像 I_gray
  │
```
